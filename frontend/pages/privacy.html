Política de Privacidad

Reglamento General de Protección de Datos (RGPD)

Responsable del tratamiento

Esta aplicación es operada por SpeakNote - Marcos Pérez.

Datos que recopilamos

Al utilizar el servicio, podemos procesar:

Archivos de audio subidos por el usuario
Transcripciones generadas
Resúmenes generados
Dirección de correo electrónico

Finalidad del tratamiento

Los datos se utilizan exclusivamente para:

Procesar los audios y generar transcripciones y resúmenes
Mejorar el funcionamiento del servicio

Base legal

El tratamiento de los datos se basa en el consentimiento del usuario al utilizar la aplicación.

Almacenamiento y servicios de terceros

Los datos se almacenan y procesan utilizando servicios de Amazon Web Services, incluyendo:

Amazon S3 (almacenamiento)
Amazon Transcribe (transcripción)
AWS Bedrock (resúmenes con IA)

Estos servicios procesan los datos únicamente para proporcionar la funcionalidad del servicio.

Retención de datos

Los datos se almacenan durante un período limitado de 48 horas y luego se eliminan automáticamente.

El usuario también puede solicitar la eliminación de sus datos en cualquier momento.

Derechos del usuario

El usuario puede solicitar:

Acceso a sus datos personales
Rectificación de sus datos personales
Eliminación de sus datos personales

Uso de datos

Los datos no se venden ni se comparten con terceros para fines comerciales.
